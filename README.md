# Gomoku

五子棋人机对战游戏 + 静态网页项目集合

## 🕹️ 项目列表

### 1. 五子棋游戏 (Gomoku)
- **文件**: `index.html`, `game.js`, `style.css`
- **功能**: 人机对战 AI 五子棋游戏
- **特性**:
  - AI 智能对手
  - 重新开始功能
  - 全屏模式
  - 胜负判定

### 2. 科技公司主页
- **文件**: `tech-company.html`
- **特性**:
  - 现代 Glassmorphism 设计
  - Bento Grid 布局
  - 响应式设计
  - Tailwind CSS

### 3. 博客页面
- **文件**: `blog.html`
- **特性**:
  - 个人博客模板
  - Tailwind CSS

### 4. 在线图书馆
- **文件**: `library.html`
- **特性**:
  - 图书馆 Landing Page
  - Tailwind CSS

## 🚀 快速开始

直接在浏览器中打开 HTML 文件：

```bash
# 打开五子棋游戏
open index.html

# 打开科技公司主页
open tech-company.html

# 打开博客页面
open blog.html

# 打开图书馆页面
open library.html

# 或使用本地服务器
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **样式框架**: Tailwind CSS (CDN)
- **无构建**: 纯静态文件
